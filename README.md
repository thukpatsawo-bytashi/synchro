# Google Sheets Dynamic Business Calendar (Apps Script)

# Overview
This project is a Google Apps Script automation that dynamically generates
a financial-year calendar (July–June) in Google Sheets.

It automatically:
- Creates month-wise sheets
- Generates calendar layout
- Highlights weekends and custom holidays
- Calculates total business days
- Updates dynamically when holidays are edited

# Features
- Financial year generation (July → June)
- Automatic holiday detection
- Business day labeling
- Real-time recalculation using onEdit trigger
- Clean automated formatting

# Tech Stack
- Google Apps Script (JavaScript)
- Google Sheets API

# Use Case
Designed for businesses to track:
- Working days
- Holidays
- Operational planning
